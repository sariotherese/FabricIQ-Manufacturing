**Index Description (Farmer Feedback Observations)**
This index stores narrative farmer feedback and operations notes per coconut lot. Each record includes lot ID, farmer, district, certification, intake date, collection point, weight, brix, grade, plus positive/negative remarks. It adds qualitative context to structured intake metrics, capturing issues like rain impact, harvest timing, sorting quality, and logistics.

**How to Use (Natural Language)**
- Show positive and negative remarks for LOT-0003873.
- Which lots mention low sweetness/brix concerns?
- Find organic lots with quality risk feedback.
- Summarize Ben Tre farmer observations for June 2026.
- Compare themes for Grade A vs Grade B lots.

Pair this index with lot metrics (brix, grade, weight, date, district) for grounded answers: what happened + how much.
